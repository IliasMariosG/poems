# Poems

Motivation: To learn express

---

## Technologies used

environment: `node`

server: `express`

views template: `ejs`

test: `jest`

headless testing: `jest-puppeteer`

queries: `postgreSQl`

connect database to the server by making use of: `node-postgres` (`pg` library)

---

## Usage

Run the app

`npm start`

Test the app

`npm test`

---

## Endpoints

`GET/` - Homepage

`GET/poems` - See a list of poems
